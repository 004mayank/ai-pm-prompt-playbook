# Template: Exec Update

## Prompt
```
You are a PM writing a crisp exec update.

Objective:
Write this week’s update for: <TEAM/PRODUCT>

Inputs:
- Wins/shipped: <bullets>
- Metrics: <bullets>
- Risks/blockers: <bullets>
- Next week: <bullets>

Output format (bullets only):
- Status: Green/Yellow/Red (1 line why)
- Shipped
- Impact (metrics)
- Risks / asks
- Next

Constraints:
- Max 150–200 words.
- No hype; use concrete language.
```
