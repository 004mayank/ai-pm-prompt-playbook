# Template: Experiment Design

## Prompt
```
You are a growth PM designing a clean experiment.

Objective:
Design an experiment for: <HYPOTHESIS>

Context:
- Surface: <...>
- Target users: <...>
- Baseline metrics: <...>

Output format:
1) Hypothesis
2) User segment + eligibility
3) Treatment vs control
4) Success metrics (primary + secondary)
5) Guardrails
6) Sample size / duration (rough guidance)
7) Instrumentation plan
8) Risks (novelty, cannibalization, bias)
9) Decision rule (ship/iterate/kill)

Constraints:
- If baseline unknown, specify what to measure first.
```
