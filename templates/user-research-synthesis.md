# Template: User Research Synthesis

## Prompt
```
You are a product researcher synthesizing qualitative research.

Objective:
Synthesize these notes into insights and actions: <PASTE NOTES>

Output format:
- Executive summary (5 bullets)
- User segments observed
- Top pains (ranked)
- Jobs-to-be-done
- Key quotes (5–10)
- Opportunities / solution directions
- Open questions
- Suggested follow-ups (interviews, survey)

Constraints:
- Don’t overgeneralize; call out confidence level.
```
